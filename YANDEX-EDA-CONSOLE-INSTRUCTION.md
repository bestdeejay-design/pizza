# 🚀 YANDEX EDA - ЗАПУСК ЗА 1 МИНУТУ

**Проблема:** Tampermonkey не работает  
**Решение:** Скрипт в консоль браузера (работает всегда!)

---

## ⚡ БЫСТРЫЙ СТАРТ

### 1. Открыть Yandex Eda
```
https://eda.yandex.ru/r/pizza_napoli_bmroq
```

### 2. Открыть консоль
Нажать **F12** → Вкладка **Console**

### 3. Скопировать скрипт
Открыть файл: https://github.com/bestdeejay-design/pizza/blob/main/yandex-eda-console-collector.js

Скопировать **ВЕСЬ код** (Ctrl+A → Ctrl+C)

### 4. Запустить
Вставить в консоль (Ctrl+V) → Нажать **Enter**

### 5. Начать сбор
Через 1 секунду появится панель справа сверху → Нажать **"🚀 Start Collection"**

---

## 📊 ЧТО ПРОИЗОЙДЁТ

1. ✅ Скрипт найдёт все продукты на странице
2. ✅ Автоматически откроет каждую модалку
3. ✅ Извлечёт данные (вес, цену, описание, ингредиенты, КБЖУ)
4. ✅ Закроет модалку
5. ✅ Перейдёт к следующему продукту
6. ✅ Сохранит JSON файл в Downloads

---

## 🎛️ ПАНЕЛЬ УПРАВЛЕНИЯ

```
┌─────────────────────────────┐
│ 🍕 Yandex Collector         │
├─────────────────────────────┤
│ Products found: 150         │
│ Collected: 45               │
├─────────────────────────────┤
│ [🚀 Start Collection]       │
│ [⏹️ Stop]                   │
└─────────────────────────────┘
```

- **Products found:** Сколько продуктов найдено
- **Collected:** Сколько уже собрано
- **Start:** Начать автоматический сбор
- **Stop:** Остановить процесс

---

## ⏱️ ВРЕМЯ СБОРА

- **1 продукт:** ~6 секунд (3 сек ожидание + 3 сек обработка)
- **150 продуктов:** ~15 минут
- **300 продуктов:** ~30 минут

Можно свернуть браузер и заниматься своими делами!

## ❓ FAQ

### Q: Скрипт не запускается?
**A:** Убедиться что страница загружена полностью (все картинки видны)

### Q: Модалки не открываются?
**A:** Проверить авторизацию в Yandex Eda

### Q: Ошибка "Cannot read property..."?
**A:** Обновить страницу и запустить скрипт заново

### Q: Можно остановить и продолжить?
**A:** Нет, но можно запустить заново - дубликаты потом удалим

### Q: Долго работает?
**A:** Это нормально - нужно открыть 150+ модалок последовательно
