# Add Weight to Products

## Usage:

```bash
cd /Users/admin/Documents/GitHub/pizza
python3 scripts/add-weight-to-products.py
```

## What it does:

- Adds `weight` field to all 333 products
- Weight ranges by category:
  - Pizza 30cm: 400-600g
  - Piccolo 20cm: 150-250g
  - Calzone: 300-500g
  - Rolls: 200-350g
  - etc.

## Result:

All products will have realistic weight values that can be adjusted later.
