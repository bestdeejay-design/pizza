## 🔧 Способы сбора

### Вариант 1: Ручной (рекомендуется для качества)
1. Откройте https://pizzanapolirsc.ru/
2. Сохраните изображения через "Сохранить изображение как..."
3. Скопируйте тексты в документы
4. Запишите цены и названия в таблицу

### Вариант 2: Автоматический (парсер)
```bash
node parser.js
```
Создаст файл `content.json` со всеми данными.

### Вариант 3: Инструменты разработчика
1. Откройте DevTools (F12)
2. Вкладка Network → отфильтруйте Images
3. Обновите страницу
4. Сохраните все URL изображений

## 📋 Структура для замены

### menuData (js/app-new.js)
```javascript
{
    id: 1,
    title: 'Маргарита',           // ← Название с сайта
    description: 'Томатный...',   // ← Состав с сайта
    price: 590,                   // ← Цена с сайта
    image: 'url...',              // ← Ссылка на фото
    category: 'pizza'             // ← Категория
}
```

### Контакты (index-new.html)
Найдите строку:
```html
<a href="tel:+79990000000">+7 (999) 000-00-00</a>
```
Замените на реальный номер с сайта.

### Зоны доставки
Найдите секцию `.delivery-card` и обновите:
- Названия районов
- Цены
- Условия бесплатной доставки

## 💡 Советы

1. **Изображения**: Используйте оригинальные URL или загрузите на хостинг
2. **Цены**: Проверьте актуальность на сайте заказчика
3. **Тексты**: Сохраняйте стиль и терминологию оригинала
4. **Акция**: Уточните условия (действительно ли 13-я бесплатно?)
