**Инструкция:**

#### Шаг 1: Установка
```
1. Открыть https://www.tampermonkey.net/
2. Скачать расширение для браузера
3. Установить
```

#### Шаг 2: Создание скрипта
```
1. Открыть Tampermonkey → Create New Script
2. Вставить код из "Метод 3" документа
3. Сохранить (Ctrl+S)
```

#### Шаг 3: Запуск
```
1. Перейти на https://eda.yandex.ru/r/pizza_napoli_bmroq
2. Авторизоваться (если нужно)
3. Скрипт запустится автоматически через 3 секунды
4. Ждать пока откроются все модалки
5. Файл yandex-eda-data.json скачается автоматически
```
