## 💡 KEY INSIGHTS

### Что узнали:

✅ **Работает:**
- Tilda API через `/tproduct/{uid}` URLs
- Text content parsing (regex) лучше CSS selectors
- Sequential modal opening с задержками
- Intermediate saving каждые 5-10 продуктов
- Retry logic для ошибок

❌ **Не работает:**
- Yandex Eda без авторизации
- Массовый быстрый парсинг
- CSS selectors для русских Tilda модалок
- Headless режим для anti-bot сайтов

⚠️ **Ограничения:**
- Tilda не хранит ингредиенты в модалках
- Yandex требует ручной вход
- Calzone, focaccia, combo sets отсутствуют в Tilda API
