## 🚀 ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС

### 1️⃣ Откройте новый сайт
```bash
open index-new.html
```
**ИЛИ** используйте Live Server в VS Code.

### 2️⃣ Соберите контент с сайта заказчика
Откройте https://pizzanapolirsc.ru/ и подготовьте:
- ✅ Названия всех пицц
- ✅ Описания (состав ингредиентов)
- ✅ Цены
- ✅ Фотографии (ПКМ → Сохранить как...)
- ✅ Телефон, адрес, время работы
- ✅ Зоны доставки и стоимость

### 3️⃣ Обновите данные в коде

**Файл `js/app-new.js`** — массив `menuData`:
```javascript
{
    id: 1,
    title: 'Маргарита',              // ← Название с сайта
    description: 'Томатный...',      // ← Состав с сайта
    price: 590,                      // ← Цена с сайта
    image: 'img/margarita.jpg',      // ← Ваше фото
    category: 'pizza'
}
```

**Файл `index-new.html`** — секция контактов:
```html
<a href="tel:+7XXXXXXXXXX">+7 (XXX) XXX-XX-XX</a>
<p>Реальный адрес</p>
```

### 4️⃣ Отправьте на GitHub
```bash
git add .
git commit -m "Update: реальный контент с pizzanapolirsc.ru"
git push origin main
```

### 5️⃣ Включите GitHub Pages
В GitHub: Settings → Pages → Source: `main` → Save

**Через 1-2 минуты сайт доступен онлайн!**

## ❓ Если что-то не так

### Сайт не открывается
Проверьте консоль браузера (F12) на ошибки.

### Не работает меню
Убедитесь, что `js/app-new.js` подключён в `index-new.html`.

### Картинки не грузятся
Проверьте пути к изображениям (должны быть относительными).

### Анимации тормозят
Отключите в браузере тяжёлые расширения.
