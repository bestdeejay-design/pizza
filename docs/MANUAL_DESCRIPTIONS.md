# 📖 Пошаговая Инструкция: Сбор Описаний Через Консоль

**Проблема:** Tilda (движок сайта) не позволяет просто так взять данные  
**Решение:** Используем ПРАВИЛЬНЫЙ скрипт для консоли

---

## ⚠️ ВАЖНО: Почему Не Работает

1. ❌ **Нельзя копировать весь файл** `scrape-descriptions.js` - он слишком большой
2. ❌ **Браузер блокирует** многострочный код
3. ❌ **Tilda использует JavaScript** - контент загружается динамически

---

## ✅ ПРАВИЛЬНЫЙ СПОСОБ

### Шаг 1: Откройте Сайт

```
1. Откройте браузер (Chrome, Safari, Firefox)
2. Перейдите на https://pizzanapolirsc.ru
3. Дождитесь полной загрузки страницы
```

### Шаг 2: Откройте Консоль Разработчика

**Windows/Linux:**
- Нажмите `F12` ИЛИ `Ctrl + Shift + J`

**Mac:**
- Нажмите `Cmd + Option + J` (Chrome/Firefox)
- ИЛИ `Cmd + Option + C` (Safari)

**Альтернатива:**
- Кликните правой кнопкой на странице → "Просмотреть код" / "Inspect"

### Шаг 3: Вставьте ЭТУ Команду

⚠️ **ВНИМАНИЕ:** Используйте ТОЛЬКО этот короткий код!

```javascript
// Скопируйте и вставьте в консоль, нажмите Enter:
document.querySelectorAll('.t-store__card').forEach((c,i) => { 
    console.log(`${i+1}. ${c.querySelector('.t-store__card__title')?.textContent || 'Без названия'} | ${c.querySelector('.t-store__card__desc')?.textContent || 'Нет описания'}`); 
});
```

Если не сработало, попробуйте альтернативу:

```javascript
// Альтернативный вариант
var p = []; document.querySelectorAll('[class*="card"]').forEach(c => { 
    var t = c.querySelector('[class*="title"], h3, h4'); 
    var d = c.querySelector('[class*="desc"], p'); 
    if(t) p.push({title: t.textContent.trim(), desc: d ? d.textContent.trim() : ''}); 
}); 
console.table(p.slice(0, 20)); 
console.log('Всего:', p.length);
```

## 💡 Советы

1. **Не пытайтесь собрать всё автоматически** - Tilda защищает от этого
2. **Лучше меньше, но точно** - 50 товаров с хорошими описаниями лучше 300 пустых
3. **Используйте копи-паст** - это быстрее чем бороться со скриптами
4. **Можно частями** - сегодня одну категорию, завтра другую

## ❓ Частые Проблемы

### "Консоль пустая после вставки"
**Решение:** Нажмите `Enter` после вставки кода

### "Ошибка синтаксиса"
**Решение:** Убедитесь что копируете **всю строку** целиком, без разрывов

### "Нашло 0 товаров"
**Решение:** 
1. Проскролльте страницу вниз до конца
2. Подождите 2-3 секунды
3. Запустите скрипт снова

### "Описания пустые"
**Причина:** На сайте нет подробных описаний  
**Решение:** Придумайте краткие (1-2 предложения) или используйте примеры выше
