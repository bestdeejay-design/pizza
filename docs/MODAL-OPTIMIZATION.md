# 🎨 Product Modal UX Optimization

**Дата:** 5 марта 2026  
**Цель:** Оптимизация продуктовой модалки для максимального удобства использования

---

## 📋 ПРОБЛЕМЫ ДО ОПТИМИЗАЦИИ

### Исходное состояние:
1. ❌ **Дублирование цены**: Цена отображалась 2 раза (в блоке и в кнопке)
2. ❌ **Вес отдельно**: Вес был в отдельном блоке, занимал много места
3. ❌ **Длинные описания**: Полные описания занимали 3-4 экрана на мобильном
4. ❌ **Неэффективное использование пространства**: Большие отступы и блоки

---

## ✅ РЕШЕНИЯ

### 1. **Inline Weight Badge** ⚖️
**Было:**
```
┌─────────────────────┐
│ Цена       │ Вес    │
│ 480 ₽      │ 480 г  │
└─────────────────────┘
```

**Стало:**
```
Пицца Маргарита [480 г]
```

**Преимущества:**
- ✅ Вес сразу виден рядом с названием
- ✅ Экономия ~30% вертикального пространства
- ✅ Визуальная связь "название-вес"
- ✅ Компактный badge стиль

**CSS:**
```css
.product-modal-title-inline-weight {
    font-size: 16px;
    font-weight: 600;
    color: var(--color-text-secondary);
    background: var(--color-bg-card);
    padding: 6px 12px;
    border-radius: 8px;
}
```

---

### 2. **Expandable Descriptions** 📝
**Было:**
- Полное описание всегда развёрнуто
- Длинные тексты занимают 3-4 экрана
- Невозможно быстро увидеть цену/кнопку

**Стало:**
```
Описание пиццы с томатами, моцареллой и базиликом...
[Показать полностью]
```

**После клика:**
```
Описание пиццы с томатами, моцареллой и базиликом, 
которое продолжает быть очень длинным и подробным,
рассказывая всю историю этого блюда...
[Свернуть]
```

**JavaScript:**
```javascript
function toggleDescription(productId) {
    const descElement = document.getElementById(`modal-desc-${productId}`);
    const isExpanded = descElement.classList.contains('expanded');
    
    if (isExpanded) {
        // Collapse - truncate to 150 chars
        descElement.innerHTML = `${fullText.substring(0, 150)}...`;
    } else {
        // Expand - show full text
        descElement.innerHTML = fullText;
    }
}
```

**Преимущества:**
- ✅ Краткая версия по умолчанию (150 символов)
- ✅ Пользователь решает, читать ли полностью
- ✅ Экономия места на мобильных
- ✅ Быстрый доступ к цене и кнопке

### 3. **Unified Price Display** 💰
**Было:**
```
┌─────────────────────┐
│ Цена: 480 ₽         │
└─────────────────────┘
...
[Добавить в корзину • 480 ₽]
```

**Стало:**
```
┌──────────────────────────────┐
│ Цена: 480 ₽                  │
└──────────────────────────────┘
...
┌──────────────────────────────┐
│ Добавить в корзину   480 ₽  │
└──────────────────────────────┘
```

**Изменения:**
- Убран дублирующий вес из price-weight-blocks
- Цена в кнопке выровнена вправо для лучшей читаемости
- Используется flexbox для распределения пространства

**CSS:**
```css
.price-weight-blocks {
    display: flex;
    align-items: center;
    gap: 16px;
    padding: 16px;
    background: var(--color-bg-card);
    border-radius: 12px;
}

.price-block {
    display: flex;
    align-items: baseline;
    gap: 8px;
}

.product-modal-add-btn {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
```

## 🎨 ВИЗУАЛЬНЫЕ СТИЛИ

- **Weight badge:** `var(--color-text-secondary)` на `var(--color-bg-card)`

### Размеры:
- **Inline weight:** 14-16px, padding 6px 12px
- **Price value:** 28px desktop, 24px mobile
- **Button:** 16px font, 18px padding

### Анимации:
- **Hover на кнопке раскрытия:** плавная смена фона (0.3s)
- **Transition всех элементов:** consistency с общим дизайном
