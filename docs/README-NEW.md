# Pizza Napoli — Сайт премиум уровня

Современный сайт пиццерии в стиле Apple/Google дизайна.

## 🎨 Дизайн-концепция

**Принципы:**
- Минимализм и чистое пространство
- Крупная типографика
- Плавные микро-взаимодействия
- Натуральные цвета (томатный, базилик, моцарелла)
- Фокус на продукте

## 📁 Структура проекта

```
pizza/

```

## 🚀 Запуск

### Вариант 1: Открыть новую версию
```bash
open index-new.html
```

### Вариант 2: Использовать Live Server (VS Code)
1. Установите расширение "Live Server"
2. Правой кнопкой на `index-new.html` → Open with Live Server

### Вариант 3: Docker
```bash
docker run --rm -d -p 8080:80 -v $(pwd):/usr/share/nginx/html nginx
```
Перейдите на http://localhost:8080

## 🎯 Функционал новой версии

### Меню
- 6 позиций с реальными фото
- Фильтрация по категориям (пицца, кальцоне, закуски)
- Анимации при наведении
- Добавление в корзину с уведомлением

### Акция
- Каждая 13-я пицца БЕСПЛАТНО
- Анимированная визуализация

### Доставка
- 3 зоны доставки
- Цветовое кодирование
- Информация о стоимости

### О нас
- Асимметричная сетка фотографий
- Ключевые преимущества
- Иконки ингредиентов

## 🔧 Следующие шаги

### Контент с сайта заказчика
Запустите парсер для сбора данных:
```bash
node parser.js
```

### Обновление данных
Отредактируйте `menuData` в файле `js/app-new.js`:
- Названия пицц
- Описания
- Цены
- Ссылки на изображения

### Деплой на GitHub Pages
```bash
git add .
git commit -m "Update: новый дизайн в стиле Apple/Google"
git push origin main
```

В настройках GitHub: Settings → Pages → Source: `main` branch
