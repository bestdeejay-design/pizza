### Вариант 1: Открыть JSON файл
```bash
# Файл с данными
menu-final.json

# Посмотреть первые 50 товаров:
head -100 menu-final.json

# Или открыть в редакторе
code menu-final.json
```

### Вариант 2: Запустить скрипт для просмотра
```bash
node view-all-descriptions.js
```

### Вариант 3: Посмотреть по категории
```javascript
// В Node.js REPL:
const m = require('./menu-final.json');

// Все товары pizza-30cm:
m.menu['pizza-30cm'].forEach(p => console.log(p.title, '-', p.description));

// Все товары с индивидуальными описаниями:
const all = Object.values(m.menu).flat();
const individual = all.filter(p => 
  !p.description.includes('Неаполитанская пицца') && 
  !p.description.includes('Маленькая персональная')
);
console.log('Индивидуальных описаний:', individual.length);
individual.forEach(p => console.log(p.title, ':', p.description));
```
