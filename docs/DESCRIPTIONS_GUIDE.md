# 📝 Инструкция по Сбору Описаний Товаров

**Дата:** 2026-03-07  
**Статус:** ⏳ ТРЕБУЕТСЯ ВРУЧНУЮ

---

## 🎯 Задача

Собрать описания товаров с сайта заказчика https://pizzanapolirsc.ru/ и добавить их в наше меню.

---

## 📋 План Действий

### Шаг 1: Посетить сайт заказчика

1. Открыть https://pizzanapolirsc.ru/
2. Прокрутить все категории товаров
3. Убедиться что товары отображаются с описаниями

---

### Шаг 2: Запустить скрипт сбора

**Вариант A: Через DevTools (рекомендуется)**

1. Открыть DevTools на сайте (**F12**)
2. Перейти во вкладку **Console**
3. Вставить код из файла `scrape-descriptions.js`
4. Нажать **Enter**
5. Скопировать результат (JSON)
6. Сохранить в файл `descriptions-temp.json`

**Вариант B: Вручную**

Если скрипт не работает, собрать данные вручную:
- Открыть каждую категорию
- Скопировать названия и описания
- Заполнить JSON формата:

```json
[
  {
    "title": "Пепперони",
    "description": "Классическая пицца с пикантной пепперони...",
    "category": "pizza-30cm"
  },
  ...
]
```

---

### Шаг 3: Объединить с нашим меню

```bash
# Запустить скрипт объединения
node merge-descriptions.js
```

**Результат:**
- Создается файл `menu-final-with-descriptions.json`
- Все найденные описания добавляются к товарам
- Отчет о найденных/не найденных совпадениях

---

### Шаг 4: Проверить результат

```bash
# Посмотреть статистику
node -e "const m = require('./menu-final-with-descriptions.json'); const all = Object.values(m.menu).flat(); console.log('С описанием:', all.filter(i => i.description).length, 'из', all.length)"
```

**Если всё правильно:**
```bash
# Заменить старый файл
mv menu-final.json menu-final-old.json
mv menu-final-with-descriptions.json menu-final.json

# Закоммитить
git add menu-final.json
git commit -m "📝 Add product descriptions from pizzanapolirsc.ru"
git push origin main
```

## 🔍 Возможные Проблемы

### ❌ Скрипт возвращает 0 товаров

**Причина:** Сайт использует динамическую подгрузку

**Решение:**
1. Проскроллить всю страницу до конца
2. Дождаться загрузки всех товаров
3. Запустить скрипт снова

ИЛИ использовать Puppeteer для автоматического скролла:

```javascript
// В DevTools Console
await new Promise(resolve => {
    let scrollHeight = document.body.scrollHeight;
    let totalScrolls = 0;
    
    const interval = setInterval(() => {
        window.scrollTo(0, document.body.scrollHeight);
        totalScrolls++;
        
        if (totalScrolls >= 5 || document.body.scrollHeight === scrollHeight) {
            clearInterval(interval);
            resolve();
        }
        
        scrollHeight = document.body.scrollHeight;
    }, 500);
});

console.log('✅ Все товары загружены, запускаем сбор...');
scrapeProductDescriptions();
```

---

### ❌ Мало совпадений (< 50%)

**Причина:** Названия отличаются

**Решение:**
1. Открыть `descriptions-temp.json`
2. Вручную проставить соответствия
3. Или использовать fuzzy search (библиотека `fuse.js`)

---

### ❌ Описания пустые или обрезанные

**Причина:** На сайте нет подробных описаний

**Решение:**
1. Проверить другие страницы (карточки товаров)
2. Использовать общие описания категорий
3. Придумать лаконичные описания самостоятельно (~50 слов)
