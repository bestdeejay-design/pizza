## 🔧 Назначение Скриптов

### collect-descriptions.js
**Цель**: Сбор описаний товаров с сайта pizzanapolirsc.ru

**Использование**:
```bash
node collect-descriptions.js
```

**Результат**: Добавляет поле `description` к товарам в JSON

---

### create-final-menu.js
**Цель**: Создание финального menu-final.json из нескольких источников

**Использование**:
```bash
node create-final-menu.js
```

**Результат**: Объединяет pizza.txt, 2.txt и другие источники

---

### download-images.sh
**Цель**: Массовое скачивание всех изображений меню

**Использование**:
```bash
chmod +x download-images.sh
./download-images.sh
```

**Результат**: Скачивает ~255 изображений в папку img/

---

### extract-images.js
**Цель**: Извлечение URL всех изображений из JSON

**Использование**:
```bash
node extract-images.js
```

**Результат**: Список URL для скачивания

---

### update-image-paths.js
**Цель**: Обновление путей к изображениям в JSON

**Использование**:
```bash
node update-image-paths.js
```

**Результат**: Заменяет абсолютные URL на относительные пути

## 💡 Когда Может Понадобиться

### 1. Добавить новые товары
- Использовать `create-final-menu.js` как шаблон
- Запустить `download-images.sh` для новых картинок

### 2. Обновить описания
- Посмотреть `collect-descriptions.js` для примера парсинга
- Адаптировать под новый сайт

### 3. Изменить структуру меню
- Взять за основу `CODE_AUDIT.md` с описанием архитектуры
- Посмотреть историю изменений в старых HTML файлах

### 4. Откатить изменения
- Если новая версия не работает - вернуть `index-old.html`
- Сравнить с текущей версией через diff
