# Pizza Napoli - Сайт доставки пиццы

**Pizza Napoli** — современный сайт доставки неаполитанской пиццы с красивым и функциональным дизайном.

## 📋 Описание проекта

Сайт-витрина (Frontend)
Красивый и функциональный сайт для заказа пиццы с каталогом продукции:
- **333 товара** в 18 категориях
- Пицца (30 см, Piccolo 20 см, Кальцоне)
- Роллы и суши
- Напитки, соусы, десерты
- Комбо-наборы
- Замороженная продукция

**Особенности:**
- Адаптивный дизайн (desktop + mobile)
- Тёмная/светлая тема
- Корзина товаров
- Навигация по категориям
- Мобильное меню

## 📊 Данные

### menu-final.json
Содержит полное меню с разбивкой по категориям:
- **metadata**: информация о проекте (источник, дата сбора, всего товаров)
- **statistics**: количество товаров по категориям
- **categories**: список категорий с описаниями
- **menu**: массив товаров по категориям

**Структура товара:**
```json
{
  "title": "Пицца Маргарита",
  "price": 650,
  "image": "img/0002-57836abce33e460be8c6f13d6c39806b.webp",
  "id": 3,
  "category": "pizza-30cm",
  "description": "Откройте для себя классику итальянской кухни...",
  "weight": 500
}
```

## 🎨 Темы оформления

Сайт поддерживает две темы:
- **Тёмная** (по умолчанию)
- **Светлая**

Переключение:
- Кнопка в хедере (desktop)
- Кнопка в мобильном меню
- Тема сохраняется в localStorage

### Для просмотра меню:
1. Откройте `index.html` в любом браузере

4. Оформите заказ
