# Pizza Napoli - Сайт доставки пиццы

**Pizza Napoli** — современный сайт доставки неаполитанской пиццы с автоматизированной системой сбора и обновления данных о продуктах.

## 📋 Описание проекта

Проект состоит из двух основных частей:

### 1. Сайт-витрина (Frontend)
Красивый и функциональный сайт для заказа пиццы с каталогом продукции:
- **333 товара** в 18 категориях
- Пицца (30 см, Piccolo 20 см, Кальцоне)
- Роллы и суши
- Напитки, соусы, десерты
- Комбо-наборы
- Замороженная продукция

**Особенности:**
- Адаптивный дизайн (desktop + mobile)
- Тёмная/светлая тема
- Корзина товаров
- Навигация по категориям
- Мобильное меню

### 2. Парсеры данных (Data Collectors)
Автоматические скрипты для сбора точных данных о продуктах из Яндекс Еды:

#### yandex-eda-auto-collector.js
Основной парсер, который:
- Открывает карточки товаров на Яндекс Еде
- Извлекает: название, вес, цену, состав, КБЖУ, описание
- Проверяет дубликаты
- Сохраняет данные в JSON

#### yandex-eda-auto-collector-clean.js
Улучшенная версия с:
- Более строгими фильтрами
- Модульной структурой
- Улучшенным поиском ингредиентов
- Автоматической валидацией данных

## 📊 Данные

### menu-final.json
Содержит полное меню с разбивкой по категориям:
- **metadata**: информация о проекте (источник, дата сбора, всего товаров)
- **statistics**: количество товаров по категориям
- **categories**: список категорий с описаниями
- **menu**: массив товаров по категориям

**Структура товара:**
```json
{
  "title": "Пицца Маргарита",
  "price": 650,
  "image": "img/0002-57836abce33e460be8c6f13d6c39806b.webp",
  "id": 3,
  "category": "pizza-30cm",
  "description": "Откройте для себя классику итальянской кухни...",
  "weight": 500
}
```

## 🔧 Настройка парсеров

### Параметры сбора
```javascript
const MAX_PRODUCTS = 2; // 2 для теста, 333 для всех товаров
```

### Что собирает парсер:
- ✅ Название продукта
- ✅ Вес (граммы)
- ✅ Цена (рубли)
- ✅ Состав (ингредиенты)
- ✅ КБЖУ (белки, жиры, углеводы, калории)
- ✅ Описание

### Алгоритм работы:
1. Находит все карточки товаров на странице
2. Извлекает название ДО открытия модалки
3. Открывает модалку каждого товара
4. Собирает полные данные
5. Закрывает модалку
6. Переходит к следующему товару
7. Сохраняет JSON с уникальными товарами

## 🤝 Как использовать

### Для просмотра меню:
1. Откройте `index.html` в любом браузере
2. Выберите категорию в sidebar
3. Добавьте товары в корзину
4. Оформите заказ

### Для обновления данных:
1. Перейдите на страницу Яндекс Еды
2. Откройте консоль браузера (F12)
3. Скопируйте код из `yandex-eda-auto-collector.js`
4. Вставьте в консоль и нажмите Enter
5. Дождитесь завершения сбора
6. Скачайте полученный JSON файл

## 📝 Примечания

- Парсер работает только на странице Яндекс Еды
- Для корректной работы убедитесь, что открыта правильная URL
- Парсер автоматически игнорирует дубликаты
- Файл с данными появится только после завершения всего процесса
