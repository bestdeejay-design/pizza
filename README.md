## 🛠️ Разработка

### Добавить новый товар
1. Добавить в `menu-final.json`
2. Положить картинку в `img/`
3. Обновить счетчик в сайдбаре

### Изменить цену
Отредактировать поле `price` в `menu-final.json`

### Поменять тему
Изменить CSS переменные в `<style>`:
```css
:root {
    --color-bg-dark: #0a0a0a;
    --color-primary: #ff2e55;
}
```
