## 🔄 Как обновить контент с сайта заказчика

### Шаг 1: Сбор данных
Откройте https://pizzanapolirsc.ru/ и соберите:
- Названия пицц
- Описания (состав)
- Цены
- Фотографии (ПКМ → Сохранить как...)
- Телефон, адрес
- Зоны доставки и цены

### Шаг 2: Обновление меню
Откройте `js/app-new.js`, найдите `menuData` и замените:

```javascript
{
    id: 1,
    title: 'Маргарита',              // ← Вставьте название
    description: 'Томатный...',      // ← Вставьте состав
    price: 590,                      // ← Вставьте цену
    image: 'photo.jpg',              // ← Вставьте путь к фото
    category: 'pizza'                // ← pizza/calzone/sides
}
```

### Шаг 3: Обновление контактов
Откройте `index-new.html`, найдите секцию `#contacts`:

```html
<a href="tel:+7XXXXXXXXXX">+7 (XXX) XXX-XX-XX</a>
<p>Адрес пиццерии</p>
<p>Часы работы</p>
```

### Шаг 4: Обновление зон доставки
Найдите секцию `#delivery` и измените карточки:

```html
<h3>Название района</h3>
<p>Доставка: XXX ₽</p>
<p>Бесплатно от: XXXX ₽</p>
```

### Шаг 5: Замена фотографий
1. Сохраните фото в папку `img/`
2. Обновите пути в HTML и JS

## 📱 Как открыть на телефоне

### Вариант 1: GitHub Pages
После деплоя откройте ссылку на телефоне.


## 💡 Советы по развитию

### Добавить новые позиции в меню
Просто добавьте объект в массив `menuData` в файле `js/app-new.js`.

### Изменить цветовую схему
В `css/style-new.css` измените переменные:
```css
:root {
    --color-primary: #ff4757;  /* Основной цвет */
    --color-accent: #ffa502;   /* Акцент */
}
```

### Добавить виджет карты
В секцию `#map` вставьте iframe Яндекс.Карт или Google Maps.

### Настроить форму заказа
Добавьте модальное окно с полями:
- Имя
- Телефон
- Адрес доставки
- Способ оплаты
